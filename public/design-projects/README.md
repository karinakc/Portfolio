# Design Projects Images

Place your graphic design project images in this folder.

## Supported formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Recommended:
- Image dimensions: 800x600px or similar aspect ratio
- File size: Keep under 500KB for faster loading
- Use descriptive names: `brand-identity.jpg`, `poster-design.png`, etc.

## Usage:
After adding an image here, reference it in your project like:
```typescript
image: "/design-projects/your-image-name.jpg"
```
